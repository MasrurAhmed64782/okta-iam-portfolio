# Project 01 — IAM Lifecycle Automation & Access Governance

## Overview

This project automates the full Joiner/Mover/Leaver (JML) identity lifecycle using **Okta Workflows** and the **Okta API**, with access governance implemented via **Okta Identity Governance (OIG)**. It mirrors enterprise-grade IAM practices used in production environments.

---

## Objectives

- Automate user onboarding (Joiner) with role-based group assignment
- Automate role-change workflows (Mover) triggered by attribute changes
- Automate offboarding and deprovisioning (Leaver)
- Generate access review reports to validate entitlements and support audit readiness

---

## Technologies Used

| Tool | Purpose |
|------|---------|
| Okta Workflows | No-code/low-code automation (JML triggers & actions) |
| Okta Groups | RBAC — role-based access control |
| Okta API | Reporting, bulk operations, audit log extraction |
| OIG Access Certifications | Formal access reviews & deprovisioning validation |
| Okta System Log | Audit trail evidence |
